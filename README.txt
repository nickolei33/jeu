Falling Sand Reforge — build "Noita-ish worldgen" (local)

Lancement (local):
- Ouvrir falling_sand_reforge/index.html dans un navigateur (Chrome/Firefox).
- Aucun build. Compatible file:// (ou via un petit serveur HTTP si tu préfères).

Contrôles:
- ZQSD / WASD / Flèches : déplacer
- Espace / ↑ : sauter / nager
- Clic gauche : lancer le sort (wand)
- Clic droit : peindre / placer la matière (brush)
- Molette : rayon brush
- Shift + Molette : changer brush
- Ctrl + Molette : changer wand
- 1..7 : sélectionner une wand
- Tab : wand suivante
- R : regen monde (nouvelle seed)
- P : pause
- F1 : debug HUD
- H : HUD
- T : aide

Notes:
- Worldgen : surface + couches + cavernes + glace/icicles + décor de fond (arbres/montagnes) + ciel (lune/étoiles).
- Température : diffusion locale + changements de phase (selon matériaux).
- Chunks : simulation "active" autour joueur/caméra, chunks inactifs ignorés.
